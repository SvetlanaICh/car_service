# Описание графического интерфейса

Приложение позволяет отслеживать текущие и уже выполненные заказы автосервиса, а также выводить статистическую инфографику.

Главное окно программы отображает заказы в виде таблицы. 
В нижней части окна можно выбрать режим просмотра таблицы (постраничный или нет).
В верхней части окна - выбор параметров отображения заказов (все заказы, сортировка, фильтрация, поиск).
В верхнем правом углу - кнопка для отображения статистики по заказам.
